## 1. 简介

PolyAnalyzer 是一款高分子材料分析桌面软件，支持以下三种分析功能：

| 功能 | 说明 | 输入格式 |
|------|------|----------|
| **GPC** | 凝胶渗透色谱分析 | `.rst` 文件 |
| **Mw** | 分子量分布分析 | `.rst` 文件 |
| **DSC** | 差示扫描量热分析 | `.txt` 文件 |

### 支持平台

| 平台 | 安装包格式 |
|------|-----------|
| Windows (x64) | `.msi` / `.exe` (NSIS 安装包) |
| Linux (x64) | `.deb` / `.AppImage` |
| macOS (ARM64/Intel) | `.dmg` |

## 2. 安装与启动

### Windows

1. 下载 `.msi` 或 `.exe` 安装包
2. 双击运行安装程序，按提示完成安装
3. 从开始菜单或桌面快捷方式启动 **PolyAnalyzer**

### Linux

**Debian/Ubuntu (deb)：**
```bash
sudo dpkg -i polyanalyzer_2.0.0_amd64.deb
```

**AppImage：**
```bash
chmod +x PolyAnalyzer_2.0.0_amd64.AppImage
./PolyAnalyzer_2.0.0_amd64.AppImage
```

### macOS

1. 下载 `.dmg` 文件
2. 双击打开，将 **PolyAnalyzer** 拖入 Applications 文件夹
3. 首次打开如提示"无法验证开发者"，前往 **系统设置 → 隐私与安全性** 点击"仍要打开"

### 目录结构

安装完成后，软件会在安装目录下自动创建以下文件夹：

```
安装目录/
├── datapath/          ← 默认数据文件夹（自动创建）
└── setting/           ← 设置文件夹（自动创建）
    ├── defaultSetting.ini       ← Mw/GPC 默认设置
    └── defaultDSCSetting.ini    ← DSC 默认设置
```

## 4. GPC 分析

### 4.1 操作步骤

1. **选择数据文件夹** — 点击"数据文件夹"旁的浏览按钮，选择包含 `.rst` 文件的目录
2. **查看文件列表** — 目录中的 `.rst` 文件将自动列出
3. **选择文件**（可选）— 勾选"选择部分文件"后，可在列表中选取需要分析的文件
4. **设置输出文件名** — 输入输出文件名（默认为当天日期，格式 `YYYYMMDD`）
5. **确认覆盖**（可选）— 如果输出目录中已有同名文件，需勾选"确认覆盖"
6. **点击"运行"** — 开始分析

### 4.2 输出结果

输出文件保存在数据文件夹的同级目录 `GPC_output/` 下：

| 文件 | 说明 |
|------|------|
| `{文件名}.csv` | 分子量汇总表（Mp, Mn, Mw, Mz, Mz+1, Mv, PD） |
| `{文件名}.png` | GPC 色谱叠加图（300 DPI） |
| `{文件名}.xlsx` | 峰数据 Excel 表格 |

### 4.3 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 保存文件 | ✅ | 输出 CSV 文件 |
| 保存图片 | ✅ | 输出 PNG 图片 |
| 保存 Excel | ✅ | 输出 XLSX 峰数据 |

## 5. Mw 分析

### 5.1 操作步骤

1. **选择数据文件夹** — 选择包含 `.rst` 文件的目录
2. **查看文件列表** — 自动列出可分析的文件
3. **设置区域分割**（可选）— 在"区域设置"中调整分子量分割位置
4. **调整画图设置**（可选）— 自定义颜色、字号、绘图元素等
5. **点击"运行"** — 开始分析，进度条显示处理进度

### 5.2 输出结果

输出文件保存在数据文件夹的同级目录 `Mw_output/` 下：

| 文件 | 说明 |
|------|------|
| `{样品名}.png` | 分子量分布图（含柱状图、Mw 曲线、分布表格） |

### 5.3 区域设置

分子量区间分割位置用于统计不同分子量范围的占比。

**默认分割位置：**
```
0, 5000, 10000, 50000, 100000, 500000, 1000000, 5000000, 10000000, 50000000
```

操作方式：
- **添加** — 输入数值后点击"添加"
- **删除** — 点击对应分割位置的删除按钮

### 5.4 画图设置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 柱状图颜色 | `#002FA7` | 分布柱状图颜色 |
| Mw 曲线颜色 | `#FF6A07` | 分子量分布曲线颜色 |
| 透明背景 | ✅ | 输出图片背景透明 |
| 绘制柱状图 | ✅ | 是否显示柱状图 |
| 绘制 Mw 曲线 | ✅ | 是否显示分子量曲线 |
| 绘制表格 | ✅ | 是否显示分布数据表格 |
| 柱状图宽度 | 1.2 | 柱子宽度 |
| Mw 曲线宽度 | 1.0 | 曲线线条宽度 |
| 坐标轴宽度 | 1.0 | 坐标轴线条宽度 |
| 标题字号 | 20 | 图表标题字体大小 |
| 坐标标题字号 | 14 | 坐标轴标签字体大小 |

## 6. DSC 分析

### 6.1 操作步骤

1. **选择数据文件夹** — 选择包含 `.txt` 文件的目录
2. **调整边界参数**（可选）— 设置左右边界和峰突出程度
3. **选择分析选项** — 勾选需要的输出类型
4. **点击"运行"** — 开始分析

### 6.2 输出结果

输出保存在数据文件夹同级目录下：

**`DSC_Cycle/` 目录：**

| 文件 | 说明 |
|------|------|
| `{样品名}.csv` | 分段数据 |
| `Cycle 1.png`, `Cycle 2.png`... | 各循环对比图 |
| `result.png` | 综合循环对比图 |

**`DSC_Pic/` 目录：**

| 文件 | 说明 |
|------|------|
| `{片段名}.png` | 各片段分析图 |

### 6.3 分析选项

| 选项 | 默认 | 说明 |
|------|------|------|
| 保存分段数据 | ✅ | 将分段数据导出为 CSV |
| 绘制分段曲线 | ✅ | 每个片段单独绘图 |
| 绘制循环对比图 | ✅ | 生成循环对比叠加图 |
| 显示对比图 | ✅ | 在界面中显示结果 |
| 保存对比图 | ✅ | 保存循环对比图为 PNG |
| 峰始终向上 | ❌ | 统一峰的方向为向上 |
| 峰居中 | ❌ | 峰位置对齐居中 |
| 测试模式 | ❌ | 调试用 |

### 6.4 边界参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 左边界 | 1.9 min | 数据左侧裁切位置 |
| 右边界 | 1.9 min | 数据右侧裁切位置 |
| 峰突出程度 | - | 峰检测灵敏度 |

### 6.5 画图设置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 曲线颜色 | `#002FA7` | DSC 曲线颜色 |
| 透明背景 | ✅ | 输出图片背景透明 |
| 曲线宽度 | 1.0 | 线条宽度 |
| 坐标轴宽度 | 1.0 | 坐标轴线条宽度 |
| 标题字号 | 20 | 图表标题字体大小 |
| 坐标标题字号 | 14 | 坐标轴标签字体大小 |

## 7. 设置管理

### 7.1 设置列表

每个分析模块都有独立的设置管理，支持多套预设方案。

- **default** — 默认设置（始终存在，不可删除）
- 用户可创建多个自定义设置方案

### 7.2 操作

| 操作 | 说明 |
|------|------|
| **切换设置** | 从下拉列表中选择已保存的设置方案 |
| **保存设置** | 输入名称后点击"保存设置"，将当前参数保存为新方案 |
| **删除设置** | 选中非默认方案后点击"删除设置" |

### 7.3 设置文件

设置文件以 `.ini` 格式保存在安装目录的 `setting/` 文件夹中：

- `defaultSetting.ini` — Mw/GPC 默认设置
- `defaultDSCSetting.ini` — DSC 默认设置
- `{自定义名称}.ini` — 用户自定义设置

## 8. 其他功能

### 8.1 清理文件夹

在"其他"标签页中，可以一键清理所有输出文件夹：

- `Mw_output/`
- `GPC_output/`
- `DSC_Cycle/`
- `DSC_Pic/`

点击"执行清理"后，上述文件夹将被清空并重新创建。

### 8.2 打开目标文件夹

分析完成后，点击"打开目标文件夹"可直接在文件管理器中打开输出目录。

### 8.3 语言切换

支持中文和英文界面，在侧边栏切换语言后立即生效。

## 9. 文件格式说明

### 9.1 GPC/Mw 输入文件（`.rst`）

- **编码：** ASCII
- **必须包含的数据段：**
  - `<MW_Averages>` ... `</MW_Averages>` — 分子量平均值
  - `<Slice_Table>` ... `</Slice_Table>` — 切片数据表

这些文件通常由 GPC 仪器软件（如 Empower）自动导出。

### 9.2 DSC 输入文件（`.txt`）

- **编码：** 文本格式
- 通常由 DSC 仪器软件导出的温度-热流数据

## 10. 常见问题

### Q: 软件启动后数据文件夹路径在哪？

A: 默认指向安装目录下的 `datapath/` 文件夹。如果该文件夹不存在，软件会自动创建。你可以通过浏览按钮手动切换到其他目录。

### Q: 分析结果保存在哪里？

A: 输出文件保存在 **所选数据文件夹的同级目录** 下。例如：

```
选择的数据文件夹: /path/to/datapath/
输出位置:
  /path/to/Mw_output/      ← Mw 分析结果
  /path/to/GPC_output/     ← GPC 分析结果
  /path/to/DSC_Cycle/      ← DSC 循环数据
  /path/to/DSC_Pic/        ← DSC 片段图片
```

### Q: 提示"存在相同文件名文件"怎么办？

A: 勾选"确认覆盖"后再次运行即可覆盖已有文件。

### Q: 设置文件丢失了怎么办？

A: 删除 `setting/` 文件夹后重启软件，会自动重新生成默认设置文件。

### Q: Linux 上 AppImage 无法运行？

A: 确保已添加执行权限：
```bash
chmod +x PolyAnalyzer_2.0.0_amd64.AppImage
```

### Q: macOS 上提示"已损坏，无法打开"？

A: 在终端中执行：
```bash
xattr -cr /Applications/PolyAnalyzer.app
```
